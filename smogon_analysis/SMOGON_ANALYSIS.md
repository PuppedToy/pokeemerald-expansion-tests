## Part 2 — Patterns by Category

### A. Setup Sweeper Patterns

1. **Quiver Dance** — Triple stat setup. Best setup move. Signal: QUIVER_DANCE in moveset. Bonus: +0.6
2. **Shell Smash** — +2 Atk/SpA/Spe. Signal: SHELL_SMASH in moveset. Bonus: +0.6
3. **Setup + Priority** — Any setup move + priority move. Signal: SETUP_MOVE + PRIORITY_MOVE. Bonus: +0.7
4. **Setup + High Speed** — Setup move + base Spe ≥ 90. After one boost, outspeeds everything. Bonus: +0.5
5. **Dragon Dance** — +1 Atk AND +1 Spe. Signal: DRAGON_DANCE in moveset. Bonus: +0.5 (less than QD/SS because doesn't boost SpA)
6. **Nasty Plot + Coverage** — NP + super-effective coverage ≥ 3 types. Bonus: +0.5
7. **Contrary + Self-Lowering Move** — Contrary + Leaf Storm/Overheat/Superpower/Draco Meteor. Signal: ability CONTRARY + those moves. Bonus: +0.7

### B. Stall / Attrition Patterns

8. **Sub + Passive Healing** — Substitute + Poison Heal / Regenerator / Leech Seed. Signal: Sub + ability/move. Bonus: +0.6
9. **Sub + Status + Recovery** — Substitute + Toxic/WoW + recovery move. Signal: three-move combo. Bonus: +0.6
10. **Hazard Setter + Recovery** — Stealth Rock or Spikes + recovery move. Signal: STEALTH_ROCK/SPIKES + RECOVERY. Bonus: +0.4
11. **Iron Barbs + Rocky Helmet interaction** — Not a moveset bonus, but Iron Barbs as ability is a strong passive (ability bonus: +0.3)

### C. Ability-Dependent Patterns

12. **Magic Guard** — Removes all residual damage drawbacks. Signal: ability MAGIC_GUARD. Bonus: +0.5
13. **Poison Heal** — Toxic Orb becomes healing. Signal: ability POISON_HEAL. Bonus: +0.5
14. **Technician + Priority** — Priority moves at ≤60 BP. Signal: TECHNICIAN + priority move ≤60 BP. Bonus: +0.5
15. **Huge Power / Pure Power** — Doubles Atk. Signal: ability HUGE_POWER or PURE_POWER. Bonus: +0.8
16. **Speed Boost + Protect** — Free +1 Spe every turn. Signal: SPEED_BOOST + Protect. Bonus: +0.6
17. **Prankster + Status** — Priority on status/hazard moves. Signal: PRANKSTER + status/hazard move. Bonus: +0.5
18. **Serene Grace + Flinch Move** — 60% flinch rate. Signal: SERENE_GRACE + Air Slash/Iron Head. Bonus: +0.5
19. **Protean / Libero** — STAB on all moves. Signal: ability check. Bonus: +0.4
20. **Skill Link + Multi-Hit** — Always 5 hits. Signal: SKILL_LINK + multi-hit move. Bonus: +0.4
21. **Mold Breaker + Ground Move** — Hits Levitate users. Signal: MOLD_BREAKER + EQ. Bonus: +0.3
22. **Unaware + Recovery** — Ignores all boosts. Signal: UNAWARE + recovery move. Bonus: +0.5
23. **Magic Bounce** — Reflects hazards/status. Signal: ability check. Bonus: +0.3
24. **Unburden + Consumable** — Double speed after item consumed. Signal: UNBURDEN + Acrobatics/berry. Bonus: +0.5

### D. Pivot / Support Patterns

25. **Pivot + Recovery** — U-turn/Volt Switch + recovery. Signal: pivot move + recovery. Bonus: +0.3
26. **Intimidate + Pivot** — Lower Atk on entry + U-turn. Signal: INTIMIDATE + U_TURN. Bonus: +0.3
27. **Regenerator + Pivot** — 33% heal on switch. Signal: REGENERATOR ability + any pivot move. Bonus: +0.4

## Part 3 — Key Principles for A6 Implementation

### Principle 1: The Tradeoff Removal Principle
The most broken combos remove a drawback that was intended to balance a strong move or mechanic:
- Magic Guard removes Life Orb recoil
- Poison Heal removes Toxic damage
- Technician removes the "priority moves are weak" tradeoff
- Contrary removes the "self-lowering moves are one-time" tradeoff
- Gale Wings (Gen 6) removed the "Flying moves have no priority" tradeoff
**Implication:** Any ability that negates a move's built-in drawback should receive a bonus.

### Principle 2: The Stacking Principle
Combos where each element makes the other stronger:
- SD + Moxie: each kill raises Atk AND SD already boosted it
- Speed Boost + Protect: free speed every turn, can't be punished
- Sub + Poison Heal: Sub costs HP but PH restores it, making Sub free
**Implication:** Synergistic combos are more than the sum of their parts. Bonus should reflect the interaction, not just each piece.

### Principle 3: The Coverage Completeness Principle
Pokemon became OU largely by having no safe switch-ins:
- Greninja: 4 different STAB types = no universal counter
- Volcarona: Fire/Bug/Grass after QD = Fire, Bug, Water, Rock, Ground coverage
- Shell Smash + Skill Link: breaks every defensive item/passive (Sash, Sub) in one hit
**Implication:** wallCount from A4 should interact with combo bonuses — a combo on a pokemon that already has 0 wallCount gets diminishing bonus.

### Principle 4: The Speed Tier Principle
Speed determines battle order, which determines everything. Combos that manipulate speed are disproportionately strong:
- Speed Boost: free +1 per turn
- Quiver Dance: +1 Spe + offense at once
- Unburden: instant double speed
- Shell Smash: +2 Spe
- Dragon Dance: +1 Spe + offense
**Implication:** Setup moves that include a Speed component (DD, QD, SS, Agility, Rock Polish) should be weighted higher than those that don't (SD, NP, CM alone).

### Principle 5: The Priority Independence Principle
Priority moves make speed checks irrelevant. Combining setup with priority creates a Pokemon that wins both the setup game AND the revenge kill game:
- After SD + Bullet Punch: can't be revenge killed by anything slower
- After BD + Aqua Jet: can't be revenge killed at all
- After setup + ExtremeSpeed: moves first against everything
**Implication:** SETUP_MOVE + PRIORITY_MOVE is the most reliable combo to detect and should receive the highest bonus.
