# T-139 — Weather + room coexistence

## Context

Deferred from T-138 (owner: "saltar de momento"). The 6v6 corpus has a MINOR pattern (Bulu/Sun TR,
Charizard-Y Sun, ~2 teams) where a weather team also runs a Trick Room MODE — the shared SLOW wallbreakers
benefit from both sun and TR moving them first. Currently a team commits to a single gimmick.

## Plan (draft)

A weather team whose abusers are SLOW (≥2 slow-strong, i.e. NOT the fast Swift-Swim/Chlorophyll speed
archetype) + a TR-move learner could splash a HALF-room TR layer (inject 1 Trick Room setter move). The open
question the owner paused on: FREQUENCY — the clean gate would also fire for slow-core weather VILLAINS (e.g.
Maxie's sun with Torkoal + Camerupt), which is corpus-legit but broad. Decide the scope (broad slow-weather /
emergent-only / bar/probability) before coding.

Acceptance criteria:

      frequency (not universal).
- [ ] Deterministic; tests + gates green.
